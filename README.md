### 색청 : 소리에 색이 보이는 공감각 (색청, 色聽, Coloured hearing)
하나의 음에서 색채를 느끼는 반응을 의미한다.

세상의 모든 소리는 색채로 표현이 됩니다. 심지어 당신의 목소리도요!
SoundBubble는 색청에서부터 아이디어를 얻어, 소리를 색으로 표현해주는 프로젝트입니다.
당신만의 소리를 세상에 하나뿐인 색으로 표현해보세요!
